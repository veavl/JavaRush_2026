# Установка Java на Windows

## Шаг 1: Скачивание JDK

1. **Откройте официальный сайт Oracle:**
   - Перейдите на https://www.oracle.com/java/technologies/downloads/
   - Или на сайт OpenJDK: https://adoptium.net/ (рекомендуется, бесплатно)

2. **Выберите версию:**
   - Рекомендуется: **Java 17 LTS** (Long Term Support) или **Java 21 LTS**
   - LTS версии поддерживаются дольше и более стабильны

3. **Выберите платформу:**
   - **Windows** → **x64 Installer** (для 64-битных систем)
   - Или **x64 ZIP Archive** (если нужна портативная версия)

## Шаг 2: Установка JDK

### Вариант A: Установка через Installer (рекомендуется)

1. Запустите скачанный `.exe` файл
2. Следуйте инструкциям мастера установки
3. **Важно:** Запомните путь установки (обычно `C:\Program Files\Java\jdk-XX`)
4. Установщик может предложить установить JRE отдельно - это не обязательно

### Вариант B: Установка из ZIP архива

1. Распакуйте архив в удобное место (например, `C:\Java\jdk-XX`)
2. Запомните путь к папке `bin` внутри распакованной директории

## Шаг 3: Настройка переменных окружения

### 3.1. Добавление JAVA_HOME

1. Откройте **Панель управления** → **Система** → **Дополнительные параметры системы**
2. Или нажмите `Win + Pause/Break` → **Дополнительные параметры системы**
3. Нажмите **Переменные среды** (Environment Variables)
4. В разделе **Системные переменные** нажмите **Создать**
5. Введите:
   - **Имя переменной:** `JAVA_HOME`
   - **Значение переменной:** путь к папке JDK (например, `C:\Program Files\Java\jdk-17`)
6. Нажмите **OK**

### 3.2. Добавление PATH

1. В том же окне **Переменные среды** найдите переменную **Path** в разделе **Системные переменные**
2. Выберите **Path** и нажмите **Изменить**
3. Нажмите **Создать** и добавьте:
   ```
   %JAVA_HOME%\bin
   ```
4. Или добавьте полный путь напрямую:
   ```
   C:\Program Files\Java\jdk-17\bin
   ```
5. Нажмите **OK** во всех окнах

## Шаг 4: Проверка установки

1. Откройте **Командную строку** (CMD) или **PowerShell**
2. Выполните команды:

```bash
java -version
```

Должно вывести что-то вроде:
```
java version "17.0.X" 2023-XX-XX LTS
Java(TM) SE Runtime Environment (build 17.0.X+X)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.X+X, mixed mode, sharing)
```

```bash
javac -version
```

Должно вывести:
```
javac 17.0.X
```

## Шаг 5: Проверка переменных окружения

В командной строке выполните:

```bash
echo %JAVA_HOME%
```

Должен вывести путь к JDK.

## Возможные проблемы и решения

### Проблема: "java не является внутренней или внешней командой"

**Решение:**
1. Убедитесь, что путь к `bin` добавлен в PATH
2. Перезапустите командную строку/PowerShell после изменения переменных окружения
3. Проверьте правильность пути к JDK

### Проблема: Установлена старая версия Java

**Решение:**
1. Проверьте, какая версия Java используется: `java -version`
2. Если старая версия, проверьте переменную PATH - возможно, там указан путь к старой версии
3. Убедитесь, что путь к новой версии JDK стоит **первым** в списке PATH

### Проблема: Несколько версий Java

**Решение:**
1. Используйте переменную `JAVA_HOME` для указания нужной версии
2. Убедитесь, что в PATH указан `%JAVA_HOME%\bin`
3. Можно использовать утилиты для переключения версий (например, jEnv для Windows)

## Альтернативные варианты установки

### Через Chocolatey (если установлен)

```bash
choco install openjdk17
```

### Через Scoop (если установлен)

```bash
scoop install openjdk17
```

### Через Winget (встроен в Windows 10/11)

```bash
winget install Microsoft.OpenJDK.17
```

## Что дальше?

После успешной установки Java вы можете:

1. **Установить IDE:**
   - IntelliJ IDEA (рекомендуется)
   - Eclipse
   - Visual Studio Code с расширением Java

2. **Создать первый проект:**
   - Создайте файл `HelloWorld.java`
   - Скомпилируйте: `javac HelloWorld.java`
   - Запустите: `java HelloWorld`

3. **Изучать Java:**
   - Используйте материалы в этой папке
   - Практикуйтесь на задачах в папках level01XX, level02XX и т.д.

## Полезные команды

```bash
# Проверка версии Java
java -version

# Проверка версии компилятора
javac -version

# Компиляция Java файла
javac MyProgram.java

# Запуск скомпилированной программы
java MyProgram

# Просмотр переменных окружения
echo %JAVA_HOME%
echo %PATH%
```

---

**Примечание:** Если вы используете IDE (IntelliJ IDEA, Eclipse), они могут автоматически обнаружить установленный JDK или предложить скачать его при первом запуске.
